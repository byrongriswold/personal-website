# Personal Portfolio Website

A personal portfolio website that showcases my background, skills, projects, and contact information. The website is built with HTML, CSS, Bootstrap, and JavaScript.

## Project Structure

```
personal-website/
├── index.html
├── contact.html
├── README.md
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── images/
    └── ...
```

## Local Setup

1. Clone or download the repository

2. Navigate to the project directory:

```
cd personal-website
```

3. Open `index.html` in a web browser.
